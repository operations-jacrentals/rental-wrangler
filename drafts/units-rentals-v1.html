1 · Hierarchy — colors · buttons · text · navigation

Everything below is already locked in (HANDOFF rules + your approved shell). Verify it reads the way you intend — this is the vocabulary the Units/Rentals build uses.

Color = meaning (one orange, one meaning)

Solid orange + dark ink — the ONE selected thing (active tab / picked option). Never used for anything else.
⛟ Linked recordOrange outline + orange ink — a LINKED record you can open. Carries the entity icon, optional ✕ unlinks.
Orange outline on a button — an armed/toggled mode (e.g. Wash mode). Hover anywhere = warm border.
Blue = commit/save. Blue text = a navigable name (hyperlink). Blue badge = informational status (Not Due).
Green = money action; green badge = good/ready/complete.
Needed Red = alert/blocked/needed; solid red button = confirm-destructive only.
ETA Jun 18Yellow = waiting/caution (End Rent, ETA pending, Incomplete).
ReservedPurple = scheduled/member states.
White + dark ink = REQUIRED until entered/captured. It stays loud until satisfied.
$1,000 totalItalic = derived/formulaic — the app computed it; you don't type it.
